## 📊 Assessment Form Overview

### What It Collects
1. **Contact Info** (Name, Email, Department)
2. **6 Assessment Questions**:
   - Q1: Coding Agents Usage (0-4)
   - Q2: LLM Knowledge (0-3)
   - Q3: Automated Usage (0-4)
   - Q4: MCP/Skills Expertise (0-4)
   - Q5: AI Expectations (multi-select)
   - Q6: Industry Awareness (0-3)

### What It Calculates
- **AI Maturity Level**: 1.0-4.5 scale
- **Levels**: Beginner, Intermediate, Advanced, Expert
- **Formula**: (Q1+Q2+Q3+Q4+Q6)/18 × 3.5 + 1.0

### What It Does
1. Validates responses (client-side)
2. Stores in localStorage (backup)
3. Shows success message
4. Resets form
5. (Optional) Creates GitHub Issue with response

**Q: Do I need to do anything after deployment?**  
A: Just check your repo for responses and share the form URL.

**Q: Can users see my private GitHub repo?**  
A: No. GitHub Pages hosts only docs/index.html publicly. Responses stay private in your repo.

**Q: What if something breaks?**  
A: Everything is self-contained. Rollback is one git revert away. Full version history.

**Q: How much does this cost?**

**Q: Can I customize the form?**
