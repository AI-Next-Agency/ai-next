## 🎯 Step 1: Enable GitHub Pages (2 minutes)

### Open GitHub Settings

1. Go to your repository on GitHub
2. Click **Settings** tab (top right)
3. Look for **Pages** in the left sidebar (under "Code and automation")

### Configure GitHub Pages

1. Under "Build and deployment" section:
   - **Source**: Click dropdown, select "Deploy from a branch"
   - **Branch**: Select `main` 
   - **Folder**: Select `/docs`

2. Click **Save**

### Wait for Build

⏳ GitHub will build your site. Wait 1-2 minutes. You'll see:
```
Your site is live at https://your-username.github.io/ai-next/
```

✅ **Done!** GitHub Pages is enabled.
