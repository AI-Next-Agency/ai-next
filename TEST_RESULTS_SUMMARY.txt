1. User opens form
2. User fills 11-question assessment
3. User clicks "Submit Assessment"
4. Form sends data to Cloudflare Worker
5. Worker validates and sanitizes input
6. Worker calls GitHub API
7. GitHub Actions workflow triggered
8. Response file created in GitHub
9. Summary document updated
10. All data committed to main branch
